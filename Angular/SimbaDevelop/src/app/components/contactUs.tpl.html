Request a Change to Techweb

To request a change to Techweb (i.e., adding, changing, or deleting a link), please contact CCHD at (505) 845-CCHD or inside-help@sandia.gov.

  • Adding Links: Provide the URL, the wording for the link, and where on Techweb you would like the new link to appear.
  • Editing / Deleting Links: Indicate where on Techweb the link you'd like changed is located and what specific changes are requested.

Other Requests

CCHD (845-CCHD or cchd@sandia.gov) can help you with:

  • Web problems (finding something on the web, using web tools or applications, writing for the web)
  • Desktop or laptop questions
  • Network, software, or hardware problems
  • Creating a collaboration site on SharePoint
