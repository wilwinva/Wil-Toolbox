How we are doing this

Sandia’s MBE goals are based on the MBE Capability Index developed by the Department of Defense Engineering Drawing Modeling Working Group (DODMWG), which is being transitioned to NIST.
Specifically, Sandia is following a phased approach to build capability to achieve Maturity Level 5 of the MBE Capability Index outlined below. By doing this Sandia will integrate MBE throughout the nuclear security complex life cycle—which should create significant efficiencies in numerous processes.

MBE Capability Index

TimeframeMaturity Level #DescriptionSummary
0 This level is the foundation upon all of the other levels are built upon. It is characterized by the dependency on the traditional 2D drawings. There is very little use if any of 3D models.
  • Drawing centric
  • Disconnected manufacturing – disconnected enterprise
  • Primary deliverable: 2d drawing
2016 1 Maturity level 1 is the first level to begin effectively using 3D models. While drawings are still the masters they are now associated to the models and both are managed together.
  • Model centric
  • Neutral Model CAM - Disconnected Enterprise
  • Primary deliverable: 2d drawing and neutral CAD model
2017 2 Maturity level 2 continues the use of 3D models with the exception of using neutral files. Instead, it centers on the reuse of the native CAD data. The models also made available to the extended enterprise by request.
  • Model centric
  • Native Model CAM - Disconnected Enterprise
  • Primary deliverable: 2d drawing and native CAD model
2018 3 This capability level is the first to consider 3D models as the master source of the product definition. At this level the drawings are considered exceptions and may contain the PMI, however, the model is the geometric definition. This level uses Product Lifecycle Management tools and 3D lightweight CAD viewables for product definition throughout the enterprise.
  • Model based definition
  • Native Model CAM - Disconnected Enterprise
  • Primary deliverable: 3d Annotated Model and lightweight viewable
2019 4 Level builds upon the level 3 capability, however, the model is now the sole master of the product definition, allowing the reuse of the model and the meta data as well.
  • Model based definition
  • Integrated manufacturing - Disconnected Enterprise
  • Primary deliverable: 3d Annotated Model and lightweight viewable
2020 5 This level is where a true Model Based Enterprise takes shape. It continues to build upon the previous levels but adds enterprise connectivity. By doing so everyone in the enterprise has up to date access to the product definition and the technical data package (TDP).
  • Model based enterprise
  • Integrated manufacturing – integrated internal enterprise
  • Primary deliverable: Digital product definition package and TDP
