Sandia Web Colors

Signature Colors

These signature colors are the primary theme colors for Sandia web applications. A suggested use of these colors would be to utilize one of the base-color blues as the primary theme color; i.e. Header background color. Then chose a different blue for components like button colors. The Orange should be reserved as an Accent color and used very sparingly for elements like: call outs to important text, main action buttons, neutral-connotation alert or notification. Tints and shades of the base colors are to be used for contrast and hierarcy; i.e button states (hover, focus), nav-bar background color, nav-bar underline border

Secondary/Background Colors

These colors broaden the color spectrum for purposes of hierarchy. Applications for these can be site-footer backgrounds, panel backgrounds, border colors, and horizontal rules.

Tertiary Colors

These colors are used primarily for content-specific needs, such as alerts and illustrations. They should never overpower the primary colors.
